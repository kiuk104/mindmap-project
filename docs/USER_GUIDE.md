# 🗺️ 마인드맵 사용설명서

라이브 사이트: **https://kiuk104.github.io/mindmap-project/**

---

## 1. 시작하기

브라우저로 접속하면 중심 주제와 4개의 샘플 노드가 보이는 마인드맵이 표시됩니다.
새로고침해도 작업 내용은 **자동 저장**되어 사라지지 않습니다.

화면 구성:
- **상단 툴바**: 주요 기능 버튼 + 검색창 + 자동저장 시각
- **캔버스**: 노드를 배치하고 연결하는 무한 작업 공간
- **하단 왼쪽**: 도움말 HUD (모바일에서는 숨김)
- **하단 오른쪽**: 현재 줌 비율

## 2. 노드 다루기

### 2.1 추가
- **Tab 키** 또는 툴바 `＋ 노드 추가` → 선택된 노드 아래에 자식 추가
- 노드 우클릭 → `➕ 자식 노드 추가`
- 배경 우클릭 → `➕ 노드 추가`

### 2.2 편집
- **더블클릭 / 더블탭** → 인라인 텍스트 편집
- **Enter** → 저장 / **Esc** → 취소
- 줄바꿈은 **Shift+Enter**

### 2.3 이동
- 노드를 마우스로 드래그하거나 손가락으로 끌어 옮김
- 부모-자식 연결선이 자동으로 따라옴

### 2.4 삭제
- 노드 선택 후 **Del / Backspace**
- 우클릭 → `🗑️ 삭제`
- ⚠️ 자식 노드도 함께 삭제되며, 해당 노드를 잇던 관계선도 자동 정리됩니다.
- ⚠️ 루트 노드는 삭제할 수 없습니다 (대신 텍스트 편집으로 의미만 바꾸기).

## 6. 검색 (Ctrl+F)

1. 툴바의 검색창에 입력 또는 **Ctrl+F**
2. 일치하는 노드는 **노란 글로우**로 강조
3. 카운트(`2 / 5`)로 매치 수 확인
4. **Enter**로 다음 매치, **Shift+Enter**로 이전 매치 — 자동으로 화면 중앙으로 스크롤
5. **Esc**로 검색 종료

## 7. 링크 추가 (4종)

노드 우클릭 → `🔗 링크 추가/관리`

| 타입 | 색상 | 미리보기 |
|---|---|---|
| 📄 구글 드라이브 문서 | 파랑 | – |
| ▶️ 유튜브 영상 | 빨강 | 마우스 호버 시 썸네일 |
| 🖼️ 이미지 URL | 초록 | 마우스 호버 시 이미지 |
| 🔗 일반 URL | 회색 | – |

링크 배지의 **×** 버튼으로 제거. 배지 클릭은 새 탭에서 열림.

## 8. 관계선 (점선 화살표)

### 만들기
1. 노드 우클릭 → `↬ 관계선 시작`
2. 시작 노드가 빨간 점선 + 깜빡임으로 강조됨
3. **다른 노드를 클릭** → 곡선 점선 화살표 생성

### 삭제
- 관계선을 클릭하면 빨간색으로 선택됨 → **Del 키**
- 또는 배경 우클릭 → `🗑️ 선택한 관계선 삭제`

### 취소
- 그리기 도중 **Esc** 또는 배경 클릭

> 💡 관계선은 부모-자식 트리 구조와 별도로, 임의의 두 노드 사이를 "참고/의존" 관계로 표현할 때 씁니다.

## 9. 저장과 불러오기

### 9.1 자동 저장 (로컬스토리지)
- **모든 변경마다 자동** (300ms 디바운스)
- 새로고침·재방문 시 그대로 복구
- 툴바 우측에 `💾 자동저장 HH:MM:SS` 표시
- ⚠️ 다른 기기/브라우저로 가면 보이지 않음 (해당 브라우저에만 저장됨)

### 9.2 JSON 파일
1. 툴바 `💾 저장` 또는 **Ctrl+S**
2. 파일 이름 입력, 형식 = `📥 JSON 파일 다운로드`
3. 다운로드된 파일을 공유 → 팀원이 `📂 불러오기`로 복구

### 9.3 클립보드 복사
저장 다이얼로그에서 형식 = `📋 클립보드에 복사` → 채팅·메모장 등에 붙여넣어 공유

### 9.4 구글 드라이브 (팀 협업)
사전 설정 필요 — `DRIVE_SETUP.md` 참조 (약 5분).

설정 후:
1. 툴바 `☁️ Drive 연결` → 구글 로그인 (한 번만)
2. 연결되면 `✅ 이메일주소`로 표시
3. 저장 시 위치를 **"구글 드라이브"** 선택 → 같은 이름이면 덮어쓰기
4. `☁️ 드라이브 불러오기` → 파일 목록에서 클릭 → 즉시 로드

> 연결은 토큰 만료 1분 전 자동 갱신되어 보통 끊김 없이 유지됩니다.
> 로그아웃하려면 `✅ 이메일주소` 버튼 다시 클릭 → 확인.

## 10. 스타일 커스터마이징

### 10.1 다크/라이트 테마
툴바 **🌓** 버튼: 다크 모드에선 ☀️, 라이트 모드에선 🌙로 표시.
- `localStorage`에 저장되어 새로고침 후에도 유지
- 처음 방문 시 시스템 설정(`prefers-color-scheme`)을 따름

### 10.2 연결선 스타일
툴바 **━ 직선** 버튼 클릭으로 순환:
- ━ 직선 → ⌒ 곡선 → ⌐ 직각 → 직선…

### 10.3 맵 스타일 (🎨 스타일 버튼)
모달에서 4가지 설정:

**색상 테마** (6종 프리셋):
| 테마 | 분위기 |
|---|---|
| Default | 균형 (빨강·파랑·보라·초록) |
| Dawn | 파스텔 핑크·살구 |
| Ocean | 청록·민트 |
| Forest | 초록·라임 |
| Sunset | 빨강·주황 |
| Mono | 그레이스케일 |

> 테마를 바꾸면 "기존 노드도 새 색으로 칠할까요?"가 뜹니다. 예 → 일괄 재칠. 아니오 → 새 노드만 새 팔레트 사용.

**배경 색**: 컬러 피커로 직접 지정. "테마 기본으로" 버튼으로 복귀.

**연결선 두께**: 얇게 / 보통 / 굵게

**자식 노드 색상으로 연결선**: 체크 시 가지마다 다른 색 (가지가 명확히 구분됨).

**노드 폰트**: 기본 / 고딕 / 명조 / 고정폭 (시스템 폰트만 사용 — 오프라인 OK)

## 11. 모바일에서 사용

화면 너비 768px 이하에선:
- 툴바 자동 줄바꿈 (2행)
- 버튼·메뉴·모달 크게 표시 (터치 친화)
- 도움말 HUD 숨김

제스처 요약:
- 한 손가락 = 이동 / 드래그
- 두 손가락 핀치 = 줌
- 길게 누름 0.5초 = 우클릭 메뉴

**홈 화면 추가** (앱처럼 쓰기):
- iOS Safari: 공유 → "홈 화면에 추가"
- Android Chrome: 메뉴 → "앱 설치" 또는 "홈 화면에 추가"
- 데스크톱 Chrome/Edge: 주소창 끝 ⬇️ 아이콘

## 12. 자주 묻는 질문 (FAQ)

**Q. 작업한 게 사라졌어요.**
A. 로컬스토리지는 브라우저별로 분리됩니다. 다른 브라우저·시크릿 모드·디바이스에서는 안 보입니다. 평소 **💾 저장**으로 JSON을 백업하거나 **Drive 연결**을 권장합니다.

**Q. 같은 마인드맵을 여러 기기에서 보고 싶어요.**
A. Drive 연결 후 같은 파일명으로 저장하면 모든 기기에서 같은 파일을 열 수 있습니다.

**Q. Drive 연결이 자꾸 끊겨요.**
A. 브라우저 쿠키/세션이 삭제되면 재로그인 필요. 일반적으로는 토큰이 자동 갱신되어 며칠~수 주 유지됩니다.

**Q. Drive 로그인 시 "이 앱은 Google에서 확인되지 않았습니다" 경고가 떠요.**
A. 본인이 만든 OAuth 클라이언트라 정상입니다. **고급 → 안전하지 않은 페이지로 이동**으로 진행.

**Q. JSON 파일을 팀원에게 보냈는데 못 열어요.**
A. `📂 불러오기`로 JSON 파일을 선택해야 합니다. 텍스트로 붙여넣기는 안 됩니다(앞으로 추가 예정).

**Q. 노드가 너무 많아 화면을 벗어났어요.**
A. 툴바 **⌖ 화면 맞춤** 클릭 → 모든 노드가 보이도록 자동 조정.

**Q. 부모-자식이 아닌 두 노드를 어떻게 잇나요?**
A. 노드 우클릭 → `↬ 관계선 시작` → 다른 노드 클릭. 점선 화살표가 생깁니다.

**Q. 색상 팔레트를 직접 추가하고 싶어요.**
A. 현재는 6종 프리셋만 지원. 개발자는 `src/utils.js`의 `COLOR_THEMES`에 새 키를 추가할 수 있습니다.
